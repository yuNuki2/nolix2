## Usage

オプションは設けていませんが、背景にライフゲームを表示させるのは容易いことです。styleにdisplay:absolute; inset:0;z-index: -1;opacity: 50を指定するだけです。

### カスタムレンダラー
いちいちcoreをインストールしなくていいように、ヘルパー関数を用意している。
